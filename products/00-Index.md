# Products — TradeOS à-la-carte tools

Each product is built and sold as a **standalone tool** (land-and-expand), modeled
on TradingView's toolset but winning on **automated, risk-gated execution**. One
folder per product; open a product's note to see scope, its TradingView
equivalent, our differentiator, and its build checklist. We tackle them one by one.

_Strategy: `docs/strategy/2026-07-09-tradingview-teardown.md` · Roadmap:
`docs/strategy/product-roadmap.md` (in the GitHub repo). Prototypes live in
`prototypes/` on branch `app-home-prototype`._
